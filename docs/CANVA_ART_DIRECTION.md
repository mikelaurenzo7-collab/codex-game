# Signal Below — Tidewalk Contact Art Direction

## Purpose

Canva is now an optional visual-art-direction surface for Autonomous Game Architect runs. It should create or refine clear visual references when that work materially improves a playable scene, HUD hierarchy, environmental storytelling, or outward-facing presentation. It does not replace Canvas implementation, deterministic game state, tests, or internal review.

## The Tidewalk contact decision

The Tidewalk decision should be readable before it is explained:

- **Lantern Tender / Lantern Line:** a warm, low lantern chain; repaired rope; clean pylon silhouettes; steady gold signal against blue-green fog. The player reads safety, settlement trust, and delayed pursuit.
- **Black-Keel Scout / Countermark Trail:** wet charcoal paint; a partially sunk skiff; broken sightlines below outer pilings; sharp red-black tide reflections. The player reads exposure, speed, and hostile attention.

Both contacts must share the same coastal environment but never blend into the same visual silhouette, location, or interaction affordance.

## Canvas implementation cues

- Use a **gold / teal / black-water** palette: lantern gold for safe commitment, sea-glass teal for navigable signal, and near-black with restrained red for hostile salvage pressure.
- Build each contact around a strong screen-space landmark: a hanging light chain for the tender, a low skiff and tar-marked piling for the scout.
- Maintain the existing ring interaction language: a thin gold or hostile-red progress arc around the active contact, plus a short plaintext label.
- Keep the background legible at laptop scale. Fine decorative detail must not compete with player, hazard, or active objective contrast.

## Canva board brief

When a run needs external visual reference, create a single widescreen Tidewalk contact board in Canva with four panels:

1. **Mood:** rain-dark coast, drowned timber, teal mist, restrained gold light.
2. **Lantern Line:** tender, lantern string, safe-return route, settlement-trust cues.
3. **Countermark Trail:** black tar symbol, half-sunk skiff, immediate pursuit, hostile attention cues.
4. **HUD and interaction:** objective ring, signal bar, and the minimum text needed for an in-world branch choice.

Avoid generic pirate imagery, high-saturation fantasy colors, or dense UI chrome. The game’s visual language is archival-survival mystery, not nautical comedy or looter-shooter spectacle.
